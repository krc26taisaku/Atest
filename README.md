# 検定対策 Ver3.3 全部入り

## データ品質改善

全1,000語を読み込み、表示前に自動点検する処理を追加しました。

### 単語欄の修正
次のようなデータは、見える単語を括弧の前だけにします。

```text
ロジスティクス(Logistics)
↓
ロジスティクス
```

括弧内の `Logistics` は削除せず、検索と自由入力の別名として保持します。

### 答えが問題文に見える問題の修正
説明文の中に答えの単語や正式名称が含まれている場合、出題時だけ答えを隠します。

```text
ロジスティクスは、原材料の調達から…
↓
原材料の調達から…
```

元の完全な説明は回答後の解説に残ります。

### 略称表示の修正
登録済み略称だけ正式名称と日本語名を表示します。
未知の英字表記に対して説明全文を正式名称として付ける不具合を修正しました。

### 維持される機能
- 略称・英語正式名称・日本語名による検索
- 略称・正式名称・日本語名の自由入力正解判定
- 単語と説明の反転
- 四択と自由入力
- 通常学習・単語試験・総合試験
- 前の問題の確認
- 予想問題

公開後はURL末尾に `?v=50` を付けて確認してください。
